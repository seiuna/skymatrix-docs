# PowderMiner
## 选项

### tool1
- 类型 `SkyBlockItemSelector`
  - 过滤器 `MiningTool`
  - 选择器 `bestMiningTool`
- 默认 `无`
- 描述 `你的挖掘工具槽位`

### too2
- 类型 `SkyBlockItemSelector`
  - 过滤器 `MiningTool`
  - 选择器 `bestMiningTool`
- 默认 `无`
- 描述 `你的开箱槽位`

## 如何开始？
1. 当按住 `attackKey` 时为开箱模式,反之为挖掘模式。
2. 开箱模式总是会优先打开理你最近的箱子。
3. 你可以设置 `MiningHelperV2` 的 `pingless` 来快速挖掘 `hard stone`
